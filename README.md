 /  ___/  | |  |/ ___\/ __ \

     \/             \/    \/ 
</pre>

### Install

```
npm install underscore.slice
```

## Usage

```javascript
var list = [1, 2, 3, 4, 5];
_.slice(list, 2); // [[1, 2], [3, 4], [5]]
_.slice(list, 4); // [[1, 2, 3, 4], [5]]
```
